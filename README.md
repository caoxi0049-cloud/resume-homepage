# 个人求职主页

这是一个静态个人求职主页模板，用来替代传统 PDF 简历。页面包含首页、关于我、工作经历、教育背景和右下角 AI 简历助手。

## 编辑个人资料

主要内容都在 `profile-data.js` 中维护：

- `basics`：姓名、职位关键词、城市、邮箱、微信、头像、简历文件
- `about`：个人简介和能力标签
- `experiences`：公司、Logo、岗位、时间、项目名称、职责概述、项目成果
- `education`：学校、学历、专业、在校时间、补充信息
- `assistant`：AI 简历助手的默认问题、关键词匹配和回答内容

中英文内容使用 `zh` 和 `en` 两套字段。页面右上角按钮可以切换语言。

## 替换素材

把真实素材放入 `assets` 文件夹：

- `avatar.jpg`：个人照片或头像
- `resume.pdf`：可下载简历
- `company-1.png`、`company-2.png`：公司 Logo

如果文件名不同，请同步修改 `profile-data.js` 中的路径。

## AI 简历助手说明

当前版本是前端本地问答，不接入外部 AI API。它会根据 `profile-data.js` 里的关键词和回答进行匹配，适合先做求职主页原型和内容展示。

后续如果需要真正的生成式问答，可以接入后端服务，把简历、项目介绍和知识库内容作为检索资料。

## 预览和部署

直接用浏览器打开 `index.html` 即可预览。部署时上传整个文件夹到 GitHub Pages、Netlify 或任意静态托管服务。
